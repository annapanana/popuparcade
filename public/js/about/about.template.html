About This Site

This site was made with love using the NEAP stack. It includes a database (PSQL) to store project information, images, tags, videos, etc. The site uses a Node server and Heroku for hosting. Most of the site assets are stored in an Google Cloud Bucket. I used Angular as a single-page front-end engine and the Bootstrap CSS framework for a responsive grid. The sourcecode for this site can be found on Github.
