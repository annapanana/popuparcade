About Anna

Anna Lotko is a full-stack web developer and UX/UI designer. She also happens to have a background in game design and development. She has worked in the entertainment industry for over 5 years with companies such as Disney and Leapfrog. She has a passion for designing playful and engaging experiences for mobile, web, desktop and experimental interfaces. While many of Anna's projects are game-related, she is currently charting new territory on the web with the Galvanize Fullstack Immersive program in Boulder, Colorado.

About This Site

This site was made with love using the NEAP stack. It includes a database (PSQL) to store project information, images, tags, videos, etc. The site uses a Node server and Heroku for hosting. Most of the site assets are stored in an Amazon S3 Bucket. I used Angular as a single-page front-end engine and the Bootstrap CSS framework for a responsive grid. The sourcecode for this site can be found on Github.
