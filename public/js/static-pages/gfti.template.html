Postcard Design

My goal for the aesthetic of the project was to create postcards that are reminscent of classic Teich postcards.

Here are few examples of the Teich cards:

This is what I ended up producing:

Due to the unpredictable nature of dynamically populating letters with images, I kept the backgrounds fairly simple. Each letter is filled with a different image, which the user selects from a gallery of search results.

Flickr API

Before this project, I had never queried an API. I used the Flickr API to source images for filling in postcard letters. This is an example view of search results from San Francisco. Users must select the number of images needed to fill each letter in the greeting.

P5.js Generative Art Library

I relied on a canvas-based JS library called p5.js for masking the images with letters and composing the postcard. This library is based off of the Processing development environment.
