Dynamic Vector Graphics: Design Process

In order to create maps that accurately represent the layout of various gardens, I used Adobe Illustrator to abstract and normalize garden layout diagrams. Because the maps needed to reflect dynamic database information, I exported SVG files of the maps, which could then be visually modified with Angular upon adding or removing produce from a plot. Each plot could include a maximum of 12 different types of produce. The variety of produce in a plot is visually via a dot diagram. Each type of pruduce had a hex color value associated with it. The colors of the dots in a plot change color depending on what is planted. The diagrams below reflect this process of data visualization.
